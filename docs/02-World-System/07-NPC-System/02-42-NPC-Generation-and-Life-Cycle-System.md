# Purpose

This document defines the complete life cycle of NPCs.

It establishes:

- birth;
- childhood;
- education;
- adulthood;
- aging;
- death;
- generational continuity.

---

# Design Philosophy

A living world requires time.

Individuals are temporary.

The world continues.

---

# Core Principle

## Every Generation Inherits The World Left Behind By The Previous One.

---

Every NPC follows:

```
Birth

↓

Childhood

↓

Adolescence

↓

Adulthood

↓

Contribution

↓

Aging

↓

Death

↓

Legacy
```
