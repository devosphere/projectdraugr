# Purpose

This document defines how NPC societies organize power, make decisions, and experience conflicts.

It establishes:

- leadership;
- governance;
- diplomacy;
- factions;
- conflicts;
- political change.

---

# Design Philosophy

Civilizations are collections of individuals with different goals.

---

# Core Principle

## Every Society Is A Balance Between Cooperation And Conflict.

People work together because they need each other.

People conflict because they want different futures.

---

# Political Structure

Political systems emerge from:

```
Population

+

Culture

+

History

+

Resources

+

Leadership
```

---

# Governance Types

Civilizations may develop different systems.

Examples:

- tribal councils;
- monarchies;
- city states;
- democratic communities;
- religious authorities;
- military governments.

---

# No Universal System

Different civilizations develop differently.

A system that works for one society may fail in another.
