# Purpose

This document defines advanced social interactions between NPCs.

It establishes:

- relationship networks;
- social influence;
- family dynamics;
- friendships;
- rivalries;
- community behavior.

---

# Design Philosophy

A society is not created by individuals existing together.

It is created by individuals affecting each other.

---

# Core Principle

## The World Is A Network Of Relationships.

---

# Social Network Model

Every NPC exists within connections.

Connections include:

```
Family

+

Friendship

+

Profession

+

Community

+

Politics

+

Culture
```

# Final Statement

The world is not alive because NPCs move.

The world is alive because NPCs matter to each other.

Every friendship.

Every conflict.

Every family.

Every memory.

The Awakened Soul does not enter a world of characters.

They enter a world of people.
