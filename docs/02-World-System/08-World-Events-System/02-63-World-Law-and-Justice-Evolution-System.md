# Purpose

This document defines how civilizations create, enforce, and evolve legal systems.

It establishes:

- laws;
- justice systems;
- courts;
- crimes;
- punishments;
- social order.

# Design Philosophy

Law is civilization's method of maintaining stability.

---

# Core Principle
