# Purpose

This document defines how natural and dynamic events occur within the world.

It establishes:

- environmental events;
- world incidents;
- event generation;
- event progression;
- event consequences;
- player interaction.

---

# Design Philosophy

The world does not wait for the player.

The world continues.

---

# Core Principle

## Events Are Consequences Of A Living Universe.

---

# Event Definition

An event is a temporary or permanent change affecting:

- individuals;
- locations;
- ecosystems;
- civilizations;
- history.

---

# Event Sources

Events originate from:

```
Natural Processes

+

Living Creatures

+

Civilization Actions

+

Unknown Phenomena

+

Player Actions
```

---

# Event Categories

Events are divided into:

1. Natural Events
2. Environmental Events
3. Biological Events
4. Social Events
5. Civilization Events
6. Unknown Events

---

# Natural Events

Events caused by planetary systems.

Examples:

- storms;
- earthquakes;
- volcanic activity;
- seasonal changes.

---

# Environmental Events

Changes affecting ecosystems.

Examples:

- forest fires;
- drought;
- flooding;
- resource depletion.

---

# Biological Events

Changes involving life.

Examples:

- migration;
- population growth;
- disease outbreaks;
- species expansion.

---

# Civilization Events

Events created by intelligent societies.

Examples:

- wars;
- discoveries;
- migrations;
- political changes.

---

# Unknown Events

Events not fully understood.

Examples:

- ancient phenomena;
- unexplained occurrences;
- supernatural events.

---

# Event Generation Model

Events follow:

```
World State

↓

Trigger Conditions

↓

Event Creation

↓

Event Development

↓

Resolution

↓

World Change
```

---

# Trigger Conditions

Events occur when conditions are met.

Examples:

A drought occurs when:

- rainfall decreases;
- temperature increases;
- water sources decline.

---

# Event Probability

Events are influenced by:

- environment;
- location;
- time;
- previous events.

---

# No Random Chaos

Events should have believable causes.

---

# Example

A forest fire may begin because:

- dry season;
- lightning;
- careless activity.

---

# Event Lifecycle

Every event has stages.

---

# Beginning

The event starts.

Example:

A storm forms.

---

# Development

The event affects the world.

Example:

Flooding damages settlements.

---

# Resolution

The event ends naturally or through intervention.

---

# Aftermath

The world changes.

Example:

New vegetation grows.

---

# Event Persistence

Some events create permanent changes.

Examples:

- destroyed settlements;
- altered landscapes;
- new ecosystems.

---

# Dynamic Event Chains

Events may create additional events.

---

# Example

Drought:

↓

Food shortage

↓

Economic decline

↓

Migration

↓

Political tension

---

# Player Influence

The player may:

- prevent;
- worsen;
- redirect;
- ignore.

---

# Example

A player builds flood protection.

Result:

Settlement survives.

---

# Ignored Event

The world continues.

Example:

A village affected by drought may eventually disappear.

---

# Event Awareness

Different entities know different things.

---

# NPC Awareness

Based on:

- location;
- observation;
- communication.

---

# Civilization Awareness

Based on:

- information networks;
- trade;
- diplomacy.

---

# Player Awareness

Based on:

- exploration;
- observation;
- reputation.

---

# Event Scale

Events exist at different levels.

---

# Local Events

Affect:

- individuals;
- small locations.

Examples:

- broken bridge;
- missing person.

---

# Regional Events

Affect:

- multiple settlements;
- ecosystems.

Examples:

- drought;
- war.

---

# Global Events

Affect:

- entire world regions.

Examples:

- climate shifts;
- major disasters.

---

# Event Interaction With NPCs

NPCs respond according to:

- personality;
- resources;
- knowledge;
- culture.

---

# Example

A storm approaches.

A prepared settlement:

- reinforces buildings.

An unprepared settlement:

- suffers losses.

---

# Event Interaction With Ecosystems

Events affect:

- populations;
- resources;
- habitats.

---

# Example

A volcanic eruption:

Before:

Forest ecosystem.

After:

New volcanic terrain.

---

# Event Memory

The world remembers major events.

Stored as:

- historical records;
- cultural stories;
- environmental changes.

---

# Historical Events

Major events become part of civilization history.

Examples:

"The Great Flood."

"The Winter Without Sun."

"The Migration Era."

---

# Event Frequency Control

The world must maintain balance.

Avoid:

- constant disasters;
- meaningless events;
- artificial chaos.

---

# Event Importance

Events are weighted by:

- impact;
- rarity;
- consequence.

---

---

## Ecosystem System

---

## NPC System

---

## Civilization System

# Developer Notes

Events should create stories naturally.

The goal:

The player should say:

"I was there when this happened."

Not:

"The game triggered another event."

---

---

## Event Has No Witness

The event still becomes history.

# Final Statement

The world is not a stage waiting for a player.

It is a universe in motion.

Storms arrive.

Forests burn.

Rivers change.

Civilizations rise.

Civilizations fall.

The Awakened Soul enters a story already happening.

They may become part of history.

But history does not require them to exist.
