# Purpose

This document defines how civilizations establish relationships, negotiate agreements, and form alliances.

It establishes:

- diplomacy;
- treaties;
- alliances;
- negotiations;
- international relationships;
- political cooperation.

---

# Design Philosophy

Civilizations survive through relationships.

---

# Core Principle

## Power Can Force Change. Diplomacy Creates Lasting Change.

Diplomacy is the process of managing relationships between groups.

- settlements;
- factions;
- civilizations;
- organizations.
