# 18.10 – Settings UI

> **Project Draugr**
>
> *The simulation remains constant. Only the player's experience may be adjusted.*

---

# Purpose

The Settings UI allows players to configure their personal game experience without affecting the simulation itself.

Settings never modify:

- world rules
- survival mechanics
- simulation difficulty
- biological systems
- time progression
- environmental behavior

Every Chronicle exists under the same immutable laws.

---

# Design Philosophy

Project Draugr follows one immutable principle:

> **Settings change the interface—not reality.**

The Settings UI exists for accessibility, comfort, and usability.

It must never provide gameplay advantages.

---

# Opening the Settings UI

The Settings UI appears only when intentionally requested.

Examples:

```text id="9kvdqe"
Open settings.

Show settings.

Settings.
```

# Interface

Controls the appearance of the user interface.

Possible settings include:

- Light Theme
- Dark Theme
- Font Size
- UI Scaling
- Panel Width
- Animation Speed
- Compact Mode

These settings affect presentation only.

---

# Accessibility

Allows players to customize readability.

Examples:

- Larger Text
- High Contrast Mode
- Dyslexia-Friendly Font
- Color Blind Support
- Reduced Motion

Accessibility settings never change gameplay mechanics.

---

# Notifications

Controls non-simulation notifications.

Examples:

- System Messages
- Friend Requests (Future)
- Multiplayer Notifications (Future)
- Update Notifications

Narration and Body HUD updates cannot be disabled.

---

# Controls

Configures interaction preferences.

Examples:

- Keyboard Shortcuts
- Menu Hotkeys
- Action Composer Behavior
- Enter-to-Send
- Tab Completion (Future)

These settings improve usability only.

---

# Account

Displays account-related information.

Examples:

```text id="j6tpug"
Account

Player Name

Account ID

Connected Email

Current Version
```

Future cloud features may appear here.

---

# Language

The interface language may be changed independently of the simulation.

Changing language does not alter:

- names
- cultures
- world lore
- geography

Only the interface text changes.

---

# Save Preferences

The player may configure interface preferences.

Examples:

- Remember Window Layout
- Remember Theme
- Remember Font Size

Chronicle progression is never affected.

---

# What Cannot Be Changed

The Settings UI intentionally excludes gameplay options such as:

- Difficulty
- Hunger Rate
- Thirst Rate
- Survival Timers
- Time Speed
- Crafting Speed
- Construction Speed
- Item Weight
- Damage Multipliers

These values are part of the world's physical laws.

Every player experiences the same simulation.

---

# Future Expansion

Future versions may include:

- Voice Accessibility
- UI Profiles
- Streaming Mode
- Screenshot Mode
- Multiplayer Chat Settings
- Audio Settings
- Localization Packs

These additions must never alter the simulation itself.

---

# Final Principle

The Settings UI exists to improve the player's interaction with Project Draugr.

It does not change reality.

The world remains identical for every Chronicle.

Only the interface adapts to the player's preferences.
