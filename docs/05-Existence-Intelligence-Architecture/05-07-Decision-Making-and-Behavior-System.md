# Purpose

The Decision-Making and Behavior System defines how entities transform:

- perception;
- memory;
- experience;
- internal states;
- environmental conditions;

into decisions and actions.

This system governs the behavior of all existence within Project Draugr.

It applies to:

- animals;
- monsters;
- humans;
- NPCs;
- civilizations;
- artificial entities;
- higher beings;
- unknown future intelligences.

---
