# Purpose

The Existence Intelligence Framework defines the universal architecture through which all forms of intelligence exist, develop, and evolve within Project Draugr.

This framework establishes a common intelligence foundation for:

- players;
- NPCs;
- humans;
- unknown races;
- creatures;
- monsters;
- animals;
- plants;
- artificial entities;
- civilizations;
- planetary consciousness;
- universal entities;
- transcendent beings.

The purpose of this framework is to ensure that all intelligence systems operate under the same fundamental principles while allowing unlimited diversity and evolution.

---

# Core Principle

Every entity follows the same fundamental rule:

> **An entity is not defined by its category. An entity is defined by its current existence state, intelligence capability, and potential evolution path.**

---

Every entity capable of intelligence follows:

# Entity Intelligence Profile (EIP)

Every entity possesses an Entity Intelligence Profile.

The EIP is the complete representation of an entity's intelligence state.

```
Entity Intelligence Profile

├── Origin
├── Existence Classification
├── Intelligence Level
├── Awareness Capability
├── Perception Capability
├── Memory Capacity
├── Learning Capability
├── Decision Complexity
├── Evolution Potential
├── Influence Capability
└── Current Development State
```

---

# 1. Origin Layer

Defines where an entity originates.

Origin does not determine final intelligence.

Possible origins include:

```
Natural Origin

Biological Origin

Artificial Origin

Constructed Origin

Dimensional Origin

Primordial Origin

Universal Origin

Unknown Origin
```

---

# Origin Principle

Origin influences:

- initial capabilities;
- limitations;
- development path.

Origin does not determine:

- destiny;
- intelligence ceiling;
- future evolution.

---

# Example

A biological creature may evolve into higher intelligence.

An artificial creation may develop consciousness.

A primordial entity may lose knowledge through transformation.
