# Purpose

The Social and Relationship Intelligence System defines how entities interact, cooperate, compete, communicate, and form meaningful connections with other existence.

This system governs:

- individual relationships;
- family structures;
- social groups;
- communities;
- civilizations;
- species interaction;
- diplomacy;
- conflict;
- alliances;
- collective intelligence.

---

# Core Principle

> **Intelligence does not exist in isolation. Existence becomes meaningful through interaction with other existence.**

An entity's development is shaped by:

```
Self

+

Others

+

Environment

=

Social Existence
```
