# 18.9 – Map UI

> **Project Draugr**
>
> *A map does not reveal the world. It records what has been seen.*

---

# Purpose

The Map UI allows the player to review geographic knowledge accumulated by the current Chronicle.

Project Draugr does **not** reveal the world map.

Every visible location must first be personally explored, observed, surveyed, or documented.

The Map UI represents memory—not omniscience.

---

# Design Philosophy

Project Draugr follows one immutable principle:

> **Unknown land does not exist on the map.**

The world always exists.

The Chronicle simply has not discovered it yet.

The Map UI never:

- reveals unexplored regions
- uncovers hidden locations
- displays points of interest automatically
- generates quest markers

The map grows naturally alongside exploration.

---

# Opening the Map UI

The Map UI appears only when intentionally requested.

Examples:

```text id="sl6v1v"
Open map.

Show map.

View map.
```

The UI never opens automatically.

---

# Exploration

The Chronicle only maps locations personally explored.

Walking through a forest gradually reveals:

- terrain
- rivers
- lakes
- cliffs
- caves
- landmarks

Areas never visited remain unknown.

---

# Unknown Regions

Unexplored regions remain blank.

Example:

```text id="bjlwm9"
██████████████
████ Unknown ████
██████████████
```

The interface never hints at what lies beyond.

---

# Current Position

The current position is shown only while the Chronicle knows where they are.

Example:

```text id="rjlwm2"
● You
```

Severe disorientation, darkness, or other future mechanics may temporarily remove this indicator.

---

# Landmarks

Landmarks appear only after being discovered.

Examples:

- River
- Waterfall
- Mountain
- Cave
- Ancient Ruins
- Giant Tree
- Village (future)
- Bridge

Landmarks remain permanently recorded after discovery.

---

# Naming Locations

Locations may initially receive temporary names.

Example:

```text id="mjlwm8"
Unnamed River
```

The Chronicle may later rename known locations.

Example:

```text id="jlwmq5"
Silver Creek
```

Personal names remain unique to that Chronicle unless shared through maps or journals.

---

# Player Notes

The Chronicle may attach notes to discovered locations.

Example:

```text id="jlwmv7"
Silver Creek

Notes

Fresh water.

Fish observed.

Good campsite.
```

Notes represent personal observations.

---

# Physical Maps

The Map UI represents the Chronicle's remembered geography.

Future systems may introduce physical maps.

Physical maps may be:

- drawn
- copied
- traded
- stolen
- lost
- destroyed

A Chronicle without a physical map still remembers previously explored regions.

Physical maps allow knowledge to survive beyond death.

---

# Accuracy

The map reflects the Chronicle's understanding.

It is not guaranteed to be perfectly accurate.

Future systems may include:

- rough sketches
- inaccurate distances
- surveying improvements
- detailed cartography

Knowledge improves through repeated exploration.

---

# Relationship with Knowledge

The Map UI records geography.

The Knowledge UI records understanding.

Example

Map

```text id="jlwmw9"
Large Cave
```

Knowledge

```text id="xjlwm3"
Large limestone cave.

Freshwater source discovered.

Possible bat habitat.
```

The two systems complement one another.

---

# Legacy

Maps become valuable historical artifacts.

Future Chronicles may discover:

- hand-drawn maps
- expedition journals
- explorer notes
- settlement charts

Knowledge of the world may spread through surviving maps rather than direct exploration.

---

# Future Expansion

The Map UI may later support:

- elevation
- contour lines
- player markers
- custom icons
- route planning
- settlements
- political borders
- shared maps
- surveying accuracy
- multiple map layers

These additions must preserve the same philosophy.

---

# Final Principle

The Map UI does not reveal the world.

It reveals only what the Chronicle has earned through exploration.

Every mountain crossed,

every river followed,

and every path remembered

becomes part of that Chronicle's understanding of the world.
